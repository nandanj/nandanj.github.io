  1. Enrich REST service Lab

  2. Add API integration using 3scale Lab

    3. The lab above is an example of how to add API integration to your service. Use the instructions below to set-up the example instead of the services referenced in the lab. Obtain the application code. Got to the directory which is the home of your Fuse foundational course labs. % cd $FUSE_FOUNDATIONAL_LABS_HOME/examples/camel-example-restlet-jdbc % mvn clean install % mvn jetty:run % curl -d "firstName=Custfirstname&lastName=Custlastname" http://localhost:8080/rs/persons/ Expose the port using ngrok: % ngrok http 8080 Before starting the lab, please set the following environment variables: $ echo 'export GUID=' >> ~/.bashrc $ echo 'export OCP_WILDCARD_DOMAIN=apps.$GUID.openshift.opentlc.com' >> ~/.bashrc $ echo 'export OCP_USER_ID=' >> ~/.bashrc

  3. Migrate service to OCP Lab

    4. In this lab you will deploy the REST service created in Lab 1 to OCP. Alternately, you can follow the instructions in the lab at the bottom of this section to deploy a Springboot or Karaf quickstart to OCP. For both scenarios, make sure that you create unique project names. Define the following environment variable:OCP_PROJECT_PREFIX=abc, where abc are your initials. afterwards, a new project is created using: oc new-project $OCP_PROJECT_PREFIX-fis-lab
